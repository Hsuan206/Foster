

# 網頁專案：寵物寄養/託管平台
人們時常因為工作繁忙、或是需要出一趟遠門無法帶著寵物一起出門，導致寵物沒有人照顧，為了解決此一問題，提供了一個寵物的寄養及託管的交流平台來幫助寵物們找到適合的保母
